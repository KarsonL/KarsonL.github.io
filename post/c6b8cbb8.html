sqlzoo-练习9——Self Join

该博文转载自我的CSDN博客：sqlzoo-练习9——Self Join（完结篇）_Karson.L的博客-CSDN博客

sqlzoo-Self Join: https://sqlzoo.net/wiki/Self_join

  • 这期主要有：
  • 自联结，“表分身”，一表多用。
    自联结后又联结其他表，一表多用，多表更多用。

1、How many stops are in the database.

2、Find the id value for the stop ‘Craiglockhart’


3、Give the id and the name for the stops on the ‘4’ ‘LRT’ service.

4、The query shown gives the number of routes that visit either London Road (149) or Craiglockhart (53). Run the query and notice the two services that link these stops have a count of 2. Add a HAVING clause to restrict the output to these two routes.

5、Execute the self join shown and observe that b.stop gives all the places you can get to from Craiglockhart, without changing routes. Change the query so that it shows the services from Craiglockhart to London Road.

  • 自联结，只有一个表，但是当两个表用。 JOIN 的两边都是同一个表，需要起别名以将其区分成两个表。也就是查了一次再查一次，获取信息列时，需要带上表别名。
  • 自联结的对应的列就是同一列，表名不同而已。

6、The query shown is similar to the previous one, however by joining two copies of the stops table we can refer to stops by name rather than by number. Change the query so that the services between ‘Craiglockhart’ and ‘London Road’ are shown. If you are tired of these places try ‘Fairmilehead’ against ‘Tollcross’

  • 和之前的JOIN 一样，还可以和其他表格一起联结。只要列对应好就行。另外的表格也可以一个当多个用，直接JOIN，起别名就行。
  • 一个表1 对应 一个表2。不能对多个表2。表1 自联结后， 再联结表2， 表11 和表21， 表12 和表22 联结。别名得对应。如题中的 ON (a.stop=stopa.id)（表a 对 表a） 和 ON (b.stop=stopb.id)（表b 对 表b）。

7、Give a list of all the services which connect stops 115 and 137 (‘Haymarket’ and ‘Leith’)

  • 在SELECT 后面加了DISTINCT， 使得重复的行都别去掉了。这个DISTINCT 是对两个列都进行区分的，不是单独某一列。所以排除掉的是两个列都相同的那些重复行。只要一列重复不会被排除。
  • 当SELECT 有两个列时， DISTINCT 不能单独对某一列 进行筛选。

8、Give a list of the services which connect the stops ‘Craiglockhart’ and ‘Tollcross’

9、Give a distinct list of the stops which may be reached from ‘Craiglockhart’ by taking one bus, including ‘Craiglockhart’ itself, offered by the LRT company. Include the company and bus no. of the relevant services.


10、Find the routes involving two buses that can go from Craiglockhart to Lochend.
Show the bus no. and company for the first bus, the name of the stop for the transfer,
and the bus no. and company for the second bus.

  • 思路：
  1. 先找到以’Craiglockhart‘为起点的车的num, company 和终点站名字 stopb.name。注意，这里都是用表b的列,因为表b示终点站。将找到的信息作为一个表bus1，暂存着。
  2. 然后再找到以’Lochend’为终点的车的num, company 和起始站名字stopb.name。注意，这里都是用表a的列,因为表a起始站。将找到的信息作为一个表bus2，暂存着。
  3. 然后将这两个表联结在一起，将表bus1 的终点站和表bus2的起始站对应起来，就可以了。
  4. 然后bus1 的num，company，和 bus2 的num, company 以及bus1 或bus2的name（中转站）就是要的答案了。
  • 因为在bus1 和 bus2 里直接用的就是 stops.name，所以之后也不用在联结stops来得到name了。
  • 一开始还以为每个表联结3次（起始-中转-终点）就可以了，但是是不行的。

这些练习题很有意思，做下来也学到了不少东西，挺有用的，很适合初学者，比看书有意思多了。当然，书也要继续看啦。之后再看一下书，看过的要记住和动手练习！
思路很重要，查询顺序很重要，语法很重要！
